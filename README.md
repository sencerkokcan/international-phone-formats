# 🌍 International Phone Formats

A comprehensive collection of international phone number formats in JSON.

## 📦 Features

- Country names
- ISO country codes (ISO2 / ISO3)
- Dial codes
- Phone number formats (masks)
- Ready-to-use JSON structure

## 🎯 Use Cases

- Dynamic phone input formatting
- Country-based validation
- International form systems
- Localization-ready applications

## 📁 Structure

data/
  phone-formats.json

## 🧠 Example

{
  "country": "Turkey",
  "short": "TR",
  "code": "+90",
  "phoneformat": "5## ### ## ##"
}

## 🚀 Usage

Can be used in:
- JavaScript
- PHP
- Mobile apps
- SaaS systems
